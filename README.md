# Ryder System (ryder-system)
Ryder System is a Fortune 500 fleet management and logistics company that provides vehicle leasing, rental, and maintenance services. Through its Developer Portal at developer.ryder.com, Ryder offers a Fleet Management API that enables lease customers to integrate fleet and location data into their business systems. The API provides access to vehicle information, location services, maintenance history, and invoice data, with API key authentication via Azure API Management.

## APIs

### Ryder Fleet Management API
The Ryder Fleet Management API enables seamless integration of fleet and location data into business systems. It provides access to Ryder fleet details including vehicle information and specifications, location data with address and service hours, historical maintenance records, and invoice details with line items and payment status. The API is designed for Ryder lease customers and requires API key authentication via the Ocp-Apim-Subscription-Key header.
